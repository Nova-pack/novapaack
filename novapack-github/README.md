# NOVAPACK - Sistema de Gestión de Albaranes

Sistema moderno multipuesto para la gestión de albaranes de transporte urgente.

## Características

- **Gestión Cloud**: Base de datos en tiempo real con Firebase Firestore.
- **Multi-Usuario**: Registro y login independiente para cada cliente.
- **Panel de Admin**: Usuario `admin@novapack.com` con control total.
- **Generación de PDF**: Impresión de albaranes y etiquetas con QR inteligente.
- **WhatsApp**: Envío directo de confirmaciones.

## Instalación

1. Clona este repositorio.
2. Abre `index.html` en tu navegador.
3. Configura las claves de Firebase en el modal de configuración.

## Tecnologías

- HTML5 / CSS3 (Diseño Responsivo)
- JavaScript (Vanilla)
- Firebase Auth & Firestore
- QRCode.js
